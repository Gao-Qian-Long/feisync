# 用户文档与配置指南

## 概述

FeiSync 插件将本地 Obsidian 笔记作为**普通文件**同步到飞书 Drive，实现备份和多端访问。

## 功能特点

- 支持多文件夹同步（可配置多个本地文件夹到不同飞书目标）
- 手动或自动同步（基于文件变化或定时）
- 以本地文件为准，单向同步
- 上传为普通文件（非云文档）
- 支持 `.feisyncignore` 忽略规则
- 用户 OAuth 授权，无需 IP 白名单

## 安装

### 通过社区插件市场

1. 打开 Obsidian 设置 → 社区插件 → 浏览
2. 搜索 "FeiSync"
3. 点击安装，然后启用

### 手动安装

1. 下载插件文件
2. 复制到 `<vault>/.obsidian/plugins/feisync/`
3. 重启 Obsidian 并启用插件

## 配置飞书应用

### 1. 创建应用

1. 访问 [飞书开放平台](https://open.feishu.cn/app)
2. 创建企业自建应用
3. 进入「凭证与基础信息」获取 App ID 和 App Secret

### 2. 添加权限

在「权限管理」页面添加以下权限：

| 权限名称 | 权限标识 |
|----------|----------|
| 查看、编辑和管理云空间 | `drive:drive` |
| 导出文档 | `drive:export:readonly` |
| 查看、评论和编辑新版文档 | `docx:document` |
| 导入文档 | `docs:document:import` |
| 导出文档 | `docs:document:export` |

### 3. 配置网页应用

1. 进入「应用功能」→「网页应用」
2. 添加网页应用
3. 重定向 URL 填写 `http://localhost:9527/callback`

### 4. 发布应用

创建版本并发布应用，权限才会生效。

## 插件设置

### 基础配置

| 字段 | 说明 |
|------|------|
| 飞书 App ID | 飞书应用的 App ID |
| 飞书 App Secret | 飞书应用的 App Secret |
| 代理服务器地址 | 用于中转 API 请求（可选） |

### 同步文件夹配置

| 字段 | 说明 |
|------|------|
| 本地路径 | 相对于仓库根目录的路径，如 `Notes` |
| 飞书目标 | 点击选择飞书云空间中的目标文件夹 |
| 映射模式 | auto=自动创建同名文件夹, custom=使用指定文件夹 |
| 启用状态 | 是否启用此映射 |

### 同步配置

| 字段 | 说明 |
|------|------|
| 自动同步 | 监听文件变化自动上传 |
| 同步间隔（分钟） | 防抖间隔 |
| 定时同步 | 按固定间隔自动同步 |
| 定时同步间隔（分钟） | 定时同步间隔，0 表示禁用 |
| 同步删除 | 本地删除时同步删除云端文件 |
| 最大并发上传数 | 同时上传的文件数 |

### 日志配置

| 字段 | 说明 |
|------|------|
| 日志级别 | DEBUG/INFO/WARN/ERROR |

## 使用方法

### 手动同步

1. 点击左侧 Ribbon 图标 → 立即同步
2. 或使用命令面板（Ctrl+P）→ `FeiSync: Sync now`

### 自动同步

1. 在设置中开启「自动同步」
2. 设置同步间隔
3. 修改文件后，自动在指定间隔后同步

### 定时同步

1. 在设置中开启「定时同步」
2. 设置同步间隔
3. 插件将按固定间隔自动执行全量同步

## 忽略规则

创建 `.feisyncignore` 文件在仓库根目录：

```
# 注释
*.log           # 忽略扩展名
node_modules/   # 忽略目录
**/.DS_Store    # 任意位置的特定文件
!important.md   # 取消忽略
```

## 同步流程

1. 扫描本地文件夹（排除忽略文件）
2. 预获取云端文件列表
3. 遍历本地文件（基于内容哈希判断是否变化）
4. 确定目标文件夹
5. 处理云端已有文件（删除旧版本）
6. 上传文件（全量或分片）
7. 保存同步记录

## 支持的文件类型

| 类型 | 扩展名 |
|------|--------|
| 文本文件 | `.md`, `.txt`, `.json`, `.yml` 等 |
| Word 文档 | `.docx`, `.doc` |
| Excel 表格 | `.xlsx`, `.xls` |
| PDF | `.pdf` |
| 图片 | `.png`, `.jpg`, `.gif` |
| 压缩包 | `.zip`, `.rar` |

## 文件大小限制

- 全量上传：单个文件 ≤20MB
- 分片上传：>20MB 文件自动分片，无上限

## 故障排除

### 测试连接失败

- 检查 App ID 和 App Secret 是否正确
- 确认网络可以访问飞书开放平台
- 如使用代理，确认代理服务器正常运行

### 文件上传失败

- 检查文件大小是否超过限制
- 确认飞书云空间充足

### 自动同步未触发

- 确认「自动同步」已开启
- 检查文件是否在配置的同步文件夹内

### 授权失败

- 确认重定向 URL 为 `http://localhost:9527/callback`
- 检查 9527 端口是否被占用
- 确认应用已发布

## 代理服务器

如果无法直接访问飞书 API，需要搭建代理服务器。详见 `plans/proxy-server-setup.md`

## 获取文件夹 Token

1. 在飞书 Drive 进入目标文件夹
2. 地址栏 URL 类似 `https://xxx.feishu.cn/drive/folder/xxx`
3. `xxx` 部分即为文件夹 Token

或点击插件设置中的「选择文件夹」按钮浏览选择。

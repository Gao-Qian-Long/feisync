# 用户文档与配置指南

## 概述

本文档旨在帮助用户安装、配置和使用 Obsidian-Flybook 插件。提供从零开始的步骤，包括飞书应用创建、插件设置、同步操作等。

## 文档结构

1. **简介**：插件功能与适用场景。
2. **安装**：通过 Obsidian 社区插件市场或手动安装。
3. **配置飞书应用**：在飞书开放平台创建应用并获取凭证。
4. **插件设置**：在 Obsidian 中配置插件。
5. **使用指南**：手动同步、自动同步、查看日志。
6. **故障排除**：常见问题与解决方法。
7. **更新日志**：版本历史。

## 详细内容

### 1. 简介

Obsidian-Flybook 插件允许您将本地 Obsidian 笔记同步到飞书 Drive，实现备份和多端访问。特点：

- 支持指定文件夹同步。
- 手动或自动同步（基于文件变化）。
- 以本地文件为准，单向同步。
- 安全认证（使用飞书开放平台凭证）。

### 2. 安装

#### 2.1 通过社区插件市场（推荐）

1. 打开 Obsidian 设置 → 社区插件 → 浏览。
2. 搜索 “Flybook” 或 “Obsidian-Flybook”。
3. 点击安装，然后启用。

#### 2.2 手动安装

1. 从 GitHub 发布页面下载 `main.js`、`manifest.json`、`styles.css`。
2. 复制到您的 Obsidian 插件文件夹：`<vault>/.obsidian/plugins/obsidian-flybook/`。
3. 重启 Obsidian，在社区插件列表中启用。

### 3. 配置飞书应用

在开始使用前，您需要在飞书开放平台创建一个企业自建应用，并开启 Drive 相关权限。

#### 3.1 创建应用

1. 访问 [飞书开放平台](https://open.feishu.cn/app)。
2. 登录您的企业账号（个人版暂不支持，需企业账号）。
3. 点击“创建企业自建应用”，填写应用名称、描述等。
4. 创建完成后，进入应用详情页。

#### 3.2 添加权限

在“权限管理”页面，为应用添加以下权限：

- `drive:drive`：查看、编辑、管理云空间中的文件。
- `drive:file`：查看、编辑、管理云空间中的文件（具体权限根据需求选择）。

#### 3.3 获取凭证

在“凭证与基础信息”页面，找到 **App ID** 和 **App Secret**，复制保存（稍后填入插件设置）。

#### 3.4 发布应用

确保应用已发布（版本管理 → 创建版本 → 申请发布）。通常需要企业管理员审核。如果是自用，可以在测试环境下使用，无需发布。

### 4. 插件设置

在 Obsidian 设置中，找到 “Flybook” 插件设置面板。

#### 4.1 凭证配置

- **飞书 App ID**：粘贴您的 App ID。
- **飞书 App Secret**：粘贴您的 App Secret。

#### 4.2 文件夹配置

- **本地同步文件夹**：相对于仓库根目录的路径，例如 `Notes`。插件将监控此文件夹下的所有文件变化。
- **飞书目标文件夹 Token**（可选）：如果您希望将文件同步到飞书Drive中某个特定的文件夹，可以填入该文件夹的 token（如何获取文件夹 token 见附录）。留空则插件会在根目录下创建名为 `ObsidianSync` 的文件夹。

#### 4.3 同步行为

- **自动同步**：启用后，当本地文件发生变化时，插件会在等待一段时间后自动上传。
- **同步间隔（分钟）**：自动同步的防抖间隔，默认 5 分钟。

#### 4.4 操作按钮

- **测试连接**：点击验证凭证是否正确，并获取访问令牌。
- **手动同步**：立即执行一次完整同步。

### 5. 使用指南

#### 5.1 首次同步

配置好凭证和文件夹后，点击“手动同步”按钮。插件会扫描本地文件夹，并将所有文件上传到飞书Drive对应的文件夹中。

#### 5.2 自动同步

开启“自动同步”后，您对本地文件的任何修改（保存）都会触发同步。请注意，频繁修改可能会导致短时间内多次同步，插件会使用防抖机制合并。

#### 5.3 查看同步状态

同步过程中，Obsidian 右下角会显示通知。同步完成后会提示成功或失败。失败时请检查错误信息。

#### 5.4 命令面板

您也可以通过命令面板（Ctrl/Cmd+P）搜索 “Flybook: Sync now” 来触发手动同步。

#### 5.5 Ribbon 按钮

插件会在左侧 ribbon 添加一个云朵图标，点击即可同步。

### 6. 故障排除

#### 6.1 测试连接失败

- 检查 App ID 和 App Secret 是否正确。
- 检查网络连接，确保可以访问 `open.feishu.cn`。
- 确认应用已添加所需权限并已发布。

#### 6.2 文件上传失败

- 文件大小是否超过 20 MB？目前插件限制单个文件 ≤20 MB。
- 飞书Drive空间是否已满？
- 查看 Obsidian 开发者控制台（Ctrl+Shift+I）是否有错误日志。

#### 6.3 自动同步未触发

- 确认“自动同步”开关已打开。
- 确认文件保存在配置的本地同步文件夹内。
- 检查防抖间隔是否设置过长。

#### 6.4 其他问题

如果遇到其他问题，请在 GitHub Issues 提交日志和复现步骤。

### 7. 附录

#### 7.1 如何获取飞书文件夹 token

1. 在飞书Drive中进入目标文件夹。
2. 浏览器地址栏的 URL 类似 `https://example.feishu.cn/drive/folder/fldcnxxxxxxxxx`。
3. `fldcnxxxxxxxxx` 即为文件夹 token。
